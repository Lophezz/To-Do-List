# Gerenciador de Tarefas 📋

Este projeto é uma aplicação de lista de tarefas (To-Do List) desenvolvida como parte das atividades práticas realizadas em aula. O objetivo principal é consolidar conceitos de lógica de programação, manipulação de dados e organização de código.

## 🚀 Funcionalidades

O sistema permite gerenciar atividades diárias através das seguintes funções:
- **Adicionar Tarefas:** Criar novos itens na lista.
- **Visualizar Lista:** Exibir todas as tarefas cadastradas.
- **Marcar como Concluída:** Alterar o status de uma tarefa para finalizada.
- **Remover Tarefas:** Excluir itens da lista.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias/ferramentas:

* **Linguagem:** Typescript, Javascript
* **Framework:** React Native
* **Banco:** MongoDB + Express
* **Ambiente:**  VS Code
* **Versionamento:** Git & GitHub
